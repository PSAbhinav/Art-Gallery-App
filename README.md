# Art Gallery App

## 📱 Overview
Art Gallery App is an Android application that allows users to browse through artworks and view details about each piece.

## ✨ Features
- Interactive image display
- Navigation between artworks
- Title and description display

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Run the application

## 📚 Learning Outcome
This project demonstrates **state management and UI updates in Jetpack Compose**.
